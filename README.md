# wfm_simulator

Simple page for running your own WFM HTML/CSS/JS files.

## Run locally

Open `/home/runner/work/wfm_simulator/wfm_simulator/index.html` in your browser.

1. Select your HTML file.
2. (Optional) Select your CSS file.
3. (Optional) Select your JS file.
4. Click **Run WFM App**.

Your app will render inside the preview frame.
